

# aungrecon
<h4>Auto Recon tool for finding SQLi,XSS,LFi,OpenRedirect</h4>


<h5>Installation</h5>

Clone the repository
```bash
git clone https://github.com/aungsanoo-usa/aungrecon.git
```
```bash
cd aungrecon
```
```bash
chmod +x install.sh
chmod +x aungrecon.sh
```
Install requirements tools
```bash
./install.sh
```
Run Script
```bash
./aungrecon.sh
```
